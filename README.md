# A TypeScript implementation of finite-state machine core.

## Development

### Install dependencies.

```
$ npm install
```

### Run units test.

```
$ npm run test
```

### Build project.

```
$ npm run build
```

## Run example

### Build example.

```
$ npm run build-example
```

### Run example.

```
$ node dist-example/<example_name>.js
```
